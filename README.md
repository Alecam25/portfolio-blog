# 🚀 Personal Portfolio Blog

Modern personal portfolio and blog built with Astro to showcase projects, technical skills, and development experience.

---

## 🌐 Overview

This project is my personal portfolio, designed to present my work in a clean, modern, and structured way. It serves as a central hub where I showcase my projects, document my learning journey, and continuously improve my frontend development and UI/UX skills.

---

## ✨ Features

- Dynamic project system using Astro Content Collections
- Modular and reusable component-based architecture
- Clean and modern UI design
- Fully responsive layout
- Fast performance and optimized loading
- Scalable content structure

---

## 🛠️ Tech Stack

- Astro
- TypeScript
- Tailwind CSS
- JavaScript
