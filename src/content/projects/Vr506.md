## Overview

VR506 Management System is a web-based administrative platform developed for a racing simulator business. The project centralizes core operational tasks in one place, making it easier to manage products, clients, events, and internal reporting workflows.

The system follows a modular structure and includes inventory control, client registration, event management, report generation, and administrative access control. It was built as a practical solution focused on organization, usability, and day-to-day business operations.

## Key Features

- Inventory management with product registration, editing, and reporting
- Client management with registration, editing, and PDF export
- Event management with creation, editing, and reporting
- Centralized reports section
- User roles and permission management
- Login system and protected access by role
- PDF export for inventory, client, and event reports

## Technical Details

This project was built using PHP with a modular structure separated into controller, model, and view layers. The interface is rendered with HTML and CSS, while data is managed through MySQL. It also integrates Dompdf to generate downloadable PDF reports directly from the system.

The main objective of VR506 was to improve operational control and reduce manual work by providing a centralized system for managing essential company information, including inventory, clients, events, and administrative permissions.
