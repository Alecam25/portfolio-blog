## Overview

Inventory Dashboard is a web-based inventory management system created to help small businesses and restaurants organize products, categories, and stock movements in a practical and visual way.

The project focuses on improving day-to-day inventory control through a dashboard-style interface, making it easier to register products, track entries and exits, and maintain a clearer view of available stock.

## Key Features

- Product management with create, edit, and delete actions
- Category organization for better inventory structure
- Stock movement tracking for incoming and outgoing items
- Dashboard-style layout for clearer data visualization
- Basic real-time inventory control
- Modular project structure for easier maintenance

## Technical Details

The system was developed using PHP and MySQL as the core backend technologies, with HTML5, CSS3, and JavaScript for the interface and user interaction. The codebase is organized into modules such as authentication, products, movements, database configuration, and shared includes, following a practical structure for real-world web applications.
