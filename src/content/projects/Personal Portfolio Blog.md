## Overview

This project is my personal portfolio and blog, designed to showcase my work, technical skills, and development experience in a clean and modern way.

It serves as a central hub where I present my projects, document my learning process, and continuously improve my frontend development and UI/UX skills.

## Key Features

- Dynamic project system using Astro content collections
- Modular and reusable component architecture
- Clean and modern UI design
- Fully responsive layout for all devices
- Fast performance and optimized loading
- Organized content structure for scalability

## Technical Details

The project was built using Astro as the main framework, leveraging its content collections feature to manage projects dynamically. The UI is styled with Tailwind CSS, and the structure follows a component-based architecture to ensure maintainability and scalability.

## My Contribution

I designed and developed the entire project, including the layout, component structure, and content system. The goal was to create a professional and scalable portfolio that reflects real-world development practices.

## Project Goal

The main goal of this project is to build a strong online presence as a developer, showcase real projects, and continuously iterate on design, performance, and user experience.
